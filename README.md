# Covid-19-part1
COVID-19に関するデータを収集するスクリプトです。

# 事前準備

PDFのダウンロードのため、Wgetコマンドを活用します。以下の手順を踏まえてWgetを利用できるようにしてください。

## MacOSの方

1. Homebrewをインストールする。(参考 : https://brew.sh/index_ja)
2. `brew install wget`を実行する。(Homebrew内へwgetをインストールする。)
